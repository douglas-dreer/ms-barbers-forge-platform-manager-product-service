# Tags

This page lists all available tags used throughout the documentation to help you find related content.

## Available Tags

The following tags are used to categorize content:

- **Configuration** - Setup and configuration guides
- **Database** - Database-related documentation
- **Docker** - Containerization and Docker setup
- **Troubleshooting** - Problem-solving guides and error resolution
- **Environment** - Environment-specific configurations (dev, staging, production)
- **Security** - Security-related configurations and best practices
- **Performance** - Performance optimization guides
- **API** - API documentation and examples

## How to Use Tags

Tags help you:

1. **Filter Content** - Use the filter controls on each page to show only content with specific tags
2. **Discover Related Topics** - Click on tags to find all related content
3. **Navigate Efficiently** - Quickly find the type of information you need

## Search Tips

- Use the enhanced search to find content across all documentation
- Combine search terms with tag filters for precise results
- Use the suggested tags for common topics
